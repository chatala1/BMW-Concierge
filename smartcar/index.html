Smartcar API Integration

This page provides information about BMW Concierge's integration with the Smartcar API.

OAuth Redirect URI

Use this redirect URI when configuring your Smartcar application:

https://chatala1.github.io/BMW-Concierge/smartcar/callback

This endpoint is configured to handle OAuth authorization callbacks from Smartcar's API.

Integration Features

BMW Concierge uses Smartcar's API to provide the following features:

  • Vehicle Information: Access basic vehicle details and specifications
  • Location Services: Retrieve current vehicle location
  • Odometer Readings: Get current mileage and trip data
  • Security Controls: Remote lock and unlock functionality
  • Climate Control: Remote climate system management

OAuth Scopes

BMW Concierge requests the following permissions from Smartcar:

  • read_vehicle_info - Basic vehicle information
  • read_location - Vehicle location data
  • read_odometer - Mileage and trip information
  • control_security - Lock/unlock controls
  • control_climate - Climate system controls

Getting Started

To integrate your vehicle with BMW Concierge:

  1. Log in to BMW Concierge using your GitHub account
  2. Navigate to the Dashboard
  3. Look for the "Vehicle Connection" section
  4. Click "Connect Vehicle" to start the Smartcar OAuth flow
  5. Authorize BMW Concierge to access your vehicle data
  6. You'll be redirected back to complete the connection

Security & Privacy

BMW Concierge takes your privacy and security seriously:

  • OAuth 2.0: Industry-standard authorization protocol
  • Minimal Data: Only requests necessary permissions
  • No Storage: Vehicle data is not permanently stored
  • Secure Transit: All data transmitted over encrypted connections
  • User Control: You can revoke access at any time

For more information about our privacy practices, please review our Privacy Policy.
